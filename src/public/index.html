HTTP Protocol Stations

Minimal demos focused on HTTP methods, headers, auth, caching, and streaming.

Stations

1) HTML GET/POST forms
2) fetch JSON POST
3) PUT mini REST API
4) Cache-Control headers
5) Chunked transfer
6) Server-Sent Events
7) Session auth + CSRF
8) JWT + rotating refresh
9) Conditional requests
10) CORS preflight
