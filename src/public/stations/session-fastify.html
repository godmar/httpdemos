Session Auth via @fastify/session

This station demonstrates the same session-cookie workflow as Station 7 (hand-rolled sessions), but uses the @fastify/session plugin instead.

Key differences: the plugin signs cookies automatically, manages the session lifecycle (create/save/destroy) for you, and provides request.session as a first-class API. request.session.destroy() clears the store entry and cookie in one call.
